# todo

vue 연습용으로 만든 심플한 todo. 투두 데이터를 저장하진 않는다. 이후에 vuex나 타입스크립트 버전도 추가할 계획이다.

### Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
